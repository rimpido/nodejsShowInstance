NodeJS - Applikation

Description

This example application was developed in nodejs.
Even though this might not look very promising at first, it's neccessary to mention that NodeJs is a back-end framework.
Indicating that NodeJs is primarly used and meant for the functionalities in the background rather than the front end design.

Next to a small NodeJs application, a simple UI-Process-Automation is demonstrated as well.
To be more specific, this tool enables you to create an automated interaction(clicking and typing) with a target page.

Note that this is an example application, by far it is not bullet proof, meaning not all potential bugs are dealt with.
